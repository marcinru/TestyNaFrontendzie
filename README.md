# TestyNaFrontendzie

Do uruchomienia aplikacji potrzebujesz mieć zainstalowany oraz uruchomiony **Docker**!

#### https://www.docker.com/products/docker-desktop

### W celu uruchomienia aplikacji:

1. Zainstaluj paczki
```
    npm install
```

2. Uruchom aplikację
```
    npm start
```

### W celu uruchomienia testów e2e:

1. Uruchom apliakcję
2. Uruchom testy
```text
    npm test:e2e
```
